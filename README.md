# Simple Sticky Notes

very simple stikcy notes.

## Features

- 6 colors sticky notes.
- drag and drop(multiple).
- change color(multiple).
- zoom in/out.
- delete(multiple)/return.
- support epic canvas.
  - fit width.
  - resizable canvas.
  - zoom in/out with selected cards as starting point.
  - tooltip.
- add line.
- add image.
- paste image.
- import/export.
